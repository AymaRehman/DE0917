# DE0917 - Programming Languages

This repository contains all the lab assignments for the DE0917 course: **Programming Languages**. Each lab folder (`Lab1`, `Lab2`, etc.) includes the original assignment details, source code, explanations of the solutions, reports, and any relevant files specific to the lab.

## Repository Structure

- **LICENSE**: Contains the open-source license for this repository.
- **README.md**: General overview of the repository.
- **.gitignore**: Global `.gitignore` for excluding IDE and OS-related files across the whole project.
- **Lab1/**: Folder for Lab 1, containing C programming files.
- **Lab2/**: Folder for Lab 2, containing Python programming files.
- Additional folders for future labs will follow a similar structure.

Each lab folder contains:
- `assignment.md`: The original assignment content provided by the instructor.
- `labX_explanation.md`: A detailed explanation of the approach, challenges, and solutions for the lab.
- `report.md`: A self-assessment report with the rubric and a summary of completed tasks.
